Teoria

El hardware son los elementos tangibles (que se pueden tocar) de un sistema microinformático mientras que el software son los elementos intangibles (no se pueden tocar).

Switch: es un dispositivo de interconexión utilizado para conectar equipos en red formando lo que se conoce como una red de área local (LAN) y cuyas especificaciones técnicas siguen el estándar conocido como Ethernet Router: es un pequeño dispositivo que crea el puente de conexión inalámbrica entre tu proveedor de servicio de Internet y los aparatos electrónicos inteligentes de casa. Permite vincular diversos dispositivos que tengan la opción de wifi integrado en su diseño, como celulares,, impresoras y televisores, entre otros aparatos.
